Important note: This app depends on external API's/services, so occasional missing or delayed data is possible.

  • Rate limits / fair-use: Open-Meteo and Nominatim may reduce the requests when too many are sent in a short period of time.
  • Non-commercial access: This app uses free/non-commercial endpoints. Heavy usage requires a paid/pro plan (Open-Meteo) or will be limited by service policies.
  • Too many rapid actions: If the user clicks buttons repeatedly (unit switch, search, “use my location”), requests may overlap and responses may arrive late or fail. Wait a few seconds between actions.
  • Temporary service issues: External services can be slow, unavailable, or return errors.
  • Geolocation limitations: Location access may be blocked/denied by default because of the browser, OS permissions, or something else, which may prevent “Use my location” from working.
  • Reverse geocoding accuracy: Nominatim may return incomplete or unexpected place names depending on the coordinates and available map data.
  • Network problems: Slow internet, offline mode or VPN/proxy issues can interrupt API calls.
  • Data gaps: Some fields may be missing in the API response for certain locations/times, so the UI may show placeholders, like "Unexpected".
  • Device compatibility:This app is developed for desktop usage and may not display correctly or function as intended on mobile devices or smaller screens. Recomended usage is between 90%-110% of browser viewport and between 24'-32'inch screens, but any other desktop resolution can be displayed well.
  • For more information about both API's, visit Open-Meteo and Nominatim.

If something fails to load, please wait a moment,or refresh the page or try again a bit later.
